Dịch vụ thanh toán sách
Dịch vụ thanh toán sách giúp người dùng thanh toán tiền sách trên 1 trang web bán sách online

Các chức năng chính:
- Thanh toán tiền sách

Công nghệ sử dụng:
-Kiến trúc hướng dịch vụ: SOA
- Ngôn ngữ sử dụng: Java
- Database: Mysql
- Server: local
